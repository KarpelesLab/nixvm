nixvm — try-before-install

Pick a statically-linked aarch64 Linux ELF and run it right here. It executes on nixvm's software syscall interpreter, compiled to WebAssembly — there is no server involved, and the binary never leaves your browser tab.

    Static binaries must be statically-linked ELF64, little-endian, aarch64
    (EM_AARCH64) executables — the same format the native
    interpreter runs (see src/vcpu/interp.rs). There is no guest
    filesystem here beyond an in-memory /, /tmp,
    /dev, /proc, /sys, so binaries that
    only touch stdout/stderr/exit work best.
